## 二、凯利公式计算

### 参数设定

基于综合概率评估：
- **b (赔率)** = 预期收益 / |最坏情况损失| = 3.65% / 15% = **0.243**
- **p (胜率)** = 25% + 43% = **68%**
- **q (败率)** = **32%**

### 凯利公式计算

```
f* = (bp - q) / b
   = (0.243 × 0.68 - 0.32) / 0.243
   = (0.165 - 0.32) / 0.243
   = -0.155 / 0.243
   = -63.8%
```

**问题**：凯利公式给出负值，说明理论最优仓位应该是0%或做空。

### 重新审视参数

让我采用看涨分析师的赔率参数（更乐观的假设）：

```
b = 12.88% / 15% = 0.86
p = 35% + 45% = 80%
q = 20%

f* = (0.86 × 0.80 - 0.20) / 0.86
   = (0.688 - 0.20) / 0.86
   = 0.488 / 0.86
   = 56.7%
```

**采用半凯利原则**：56.7% × 50% = **28.35%**
