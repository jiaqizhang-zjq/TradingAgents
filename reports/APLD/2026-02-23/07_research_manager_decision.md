## 三、凯利公式计算

### 参数设定（基于我的概率分布）

- b = 赔率 = 1.0（上涨40%时的收益/下跌23%时的损失比）
- p = 胜率 = 0.25 + 0.40 = 0.65（看涨+基准情况的胜率）
- q = 败率 = 0.35

### 凯利公式计算

```
f* = (bp - q) / b
f* = (1.0 × 0.65 - 0.35) / 1.0
f* = (0.65 - 0.35) / 1.0
f* = 30%
```

**凯利公式最优仓位：30%**

但考虑到：
1. 单只股票不应超过组合20%（分散风险原则）
2. 高波动性成长股需要额外风险控制
3. 凯利公式在实践中应减半使用（半凯利）

**建议仓位：15%（符合半凯利原则）**
