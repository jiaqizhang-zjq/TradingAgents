## 三、关于“凯利仓位1.8%”的教条主义

保守型分析师写道：

> 凯利公式仓位仅1.8%，数学上不支持建仓

**我的反驳：**

你对凯利公式的理解**过于机械化**。

### 3.1 凯利公式的适用边界

经典凯利公式：
```
f* = (bp - q) / b
```

其中：
- b = 净赔率
- p = 胜利概率
- q = 失败概率 = 1-p

但这个公式有几个**关键假设**：
1. 收益分布已知
2. 可以无限次交易
3. 没有尾部风险

**TSLA的特性**：
- 收益分布**尖峰厚尾**
- **黑天鹅事件**频繁（对你来说是风险，对我来说是不对称收益的来源）

### 3.2 激进型凯利调整

我的做法是**半凯利 + 动态调整**：

| 条件 | 仓位调整 |
|------|----------|
| 极端超卖 + 技术底背离 | 1.5倍凯利 |
| 基本面恶化但估值回归 | 1倍凯利 |
| 趋势确认后的回调 | 0.5倍凯利 |

**当前场景**：极端超卖 + 技术底背离 + 支撑强劲 = **应该是1.5倍凯利 = 2.7%**
