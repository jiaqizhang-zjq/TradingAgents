**简化凯利公式**：
```
f* = 预期收益率 / 最大可能亏损
f* = 7.23% / 13.1% = 55.2%
```

**完整凯利公式**：
```
f* = (bp - q) / b = (0.55 × 0.52 - 0.48) / 0.55 = 18.9%
```

**错误**：过早抄底，认为"基本面稳健"
**教训**：技术面和资金面的警示先于基本面反应

**错误**：在180美元看空，认为估值太高
**结果**：涨到400美元
**教训**：不要与强势趋势对抗，但要在高位警惕反转信号
