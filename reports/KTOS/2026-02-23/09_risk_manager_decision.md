## 三、凯利公式计算

### 3.1 参数设定

基于综合分析：
- 胜率（上涨概率）= 22% + 40%×50% = 42%
- 败率（下跌概率）= 58%
- 盈亏比（b）= 潜在上涨/潜在下跌 = 27.2%/16.2% ≈ 1.68

### 3.2 凯利公式计算

**f* = (bp - q) / b**
- b = 1.68
- p = 0.42
- q = 0.58
- f* = (1.68 × 0.42 - 0.58) / 1.68
- f* = (0.7056 - 0.58) / 1.68
- f* = 0.1256 / 1.68 = **7.5%**

4. **技术面确认下降趋势**：20日均线斜率-0.927，MACD零轴下方

### 9.2 为什么不是激进做空（15%-20%仓位）

1. **Q4财报不确定性**：无论超预期还是不及预期，都是尾部风险

3. **历史准确率**：激进派过去66.7%错误率 vs 保守派25%错误率
4. **凯利公式最优仓位仅7.5%**：不支持15%-20%激进仓位

### 9.3 为什么不是保守等待（2.5%-5%仓位）

1. **等待可能错失机会**：$98-$105反弹可能永远不会来
2. **2.5%仓位收益太低**：跌16%只赚0.4%，不够交易成本
